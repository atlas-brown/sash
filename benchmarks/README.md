# Legend

## Filename format:
- `source-organization-repository-description.sh`
- Organization and repository don't apply to web forum bugs (such as StackOverflow)

## Sources:
- High Profile bugs: `hp`
- StackOverflow: `so`
- Unix & Linux Stack Exchange: `use`
- GitHub: `gh`
- GitLab: `gl`
- OpenDev: `od`

## Misc
- Inside every file the link to the original source can be found right after the shebang line
- To see the in-scope bug(s) in every file, search for the phrase `bug here`
